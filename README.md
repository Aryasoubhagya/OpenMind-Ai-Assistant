# OpenMind AI Assistant 🤖

OpenMind AI Assistant is an open-source AI chatbot framework built with Python.

The goal of this project is to help developers, students, and AI enthusiasts understand how modern chatbot systems are designed.

## Features

✅ Modular AI chatbot architecture  
✅ Conversation memory system  
✅ Custom system prompts  
✅ Easy API integration  
✅ Beginner-friendly codebase  
✅ Developer focused structure  


## Project Structure

OpenMind-AI-Assistant/

├── app.py

├── chatbot/

│   ├── ai_engine.py

│   ├── memory.py

│   └── prompts.py

├── requirements.txt

└── README.md


## Installation

Clone the repository:

git clone your-repository-link


Install dependencies:

pip install -r requirements.txt


Run:

python app.py
